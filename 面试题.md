## 常见面试题

### JS

1. 下面代码输出什么？

   ```js
   function say(){
       console.log(a);
       console.log(b);
       var a = 'ni';
       let b = 'ta';
   }
   变量作用域、变量提升问题 (ES6、ES5)
   ```

2. 下面代码输出什么？

   ```js
   const person {
       name: '张三',
       myName(){
           console.log(this.name);
       },
       Name: ()=> console.log(this.name);
   }
   person.myName();
   person.Name();
   this指向问题，箭头函数this指向问题
   ```

3. 下面代码输出什么？

   ```js
   let a = {
       greeting: '你好'
   };
   let b;
   b = a;
   a.greeting = '我不好';
   console.log(b.greeting);
   引用类型与基本类型的区别
   ```
